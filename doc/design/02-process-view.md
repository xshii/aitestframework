# AI测试框架 - 进程视图 (Process View)

## 概述

进程视图描述系统的运行时行为，关注并发、同步、通信和进程间交互。本视图展示测试框架在运行时的动态行为和进程模型。

### 1.2 进程职责划分

| 进程/线程 | 职责 | 通信方式 |
|-----------|------|----------|
| **Main Process** | 协调调度、配置管理、结果汇总 | 直接调用 |
| **Discovery Thread** | 测试用例发现和收集 | 内部队列 |
| **Reporter Thread** | 实时报告更新、日志记录 | 事件订阅 |
| **Worker Process** | 测试执行、模型推理 | IPC Queue |
| **GPU Process** | GPU资源管理、推理加速 | CUDA IPC |

### 6.2 死锁预防

├─────────────────────────────────────────────────────────────────────────────┤
